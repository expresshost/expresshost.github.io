new Express host or NEH

Is a tool genarated by SamuelFischerCode to genrate hosts for deployment with node.js

REQUIREMENTS

  • NPM
  • BASH, ZSH or any type of UNIX shell

How to install

  • Run sudo curl https://expresshost.github.io/install.sh -o install.sh; ./install.sh; rm install.sh;
  • Boom

How to use

  • Run npm run start or run npm run dev if you want to edit serve.js
  • Boom your app is deployed and online on port 8080 or a exported $PORT value
